# Welcome to the Shelf Documentation 📚

Shelf is a modern, type-safe book management system designed for power users who want a clean, fast, and functional way to organize their digital library.

## 🏁 Getting Started

If you are new to Shelf, we recommend following these guides in order:

1. **[Deployment Guide](Deployment-Guide.md)**: Set up Shelf on your server using Docker.
2. **[User Manual](User-Manual.md)**: Learn how to import books, manage metadata, and connect your reading devices.

## 📐 Architecture

Shelf is built with a focus on **Functional Domain-Driven Design (DDD)**. 

- For developers interested in the internals, please see the [Architecture and AGENTS guide](../ARCHITECTURE.md).
- Detailed domain models and primitive types are documented in the [API Reference](API-Reference.md) (coming soon).
