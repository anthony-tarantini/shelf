# Reading

Shelf includes built-in readers for both ebooks and audiobooks, accessible directly from the web UI.

## EPUB Reader

Open any EPUB book from the catalog to launch the in-browser reader. The reader supports standard navigation, bookmarking, and responsive layout across devices.

## Audiobook Player

Shelf provides a web-based audio player for audiobooks with:

- **Track navigation** — Jump between chapters and tracks.
- **Playback speed control** — Adjust listening speed to your preference.
- **Resume playback** — Pick up where you left off across sessions.

## Device Sync

For reading on dedicated e-reader hardware, Shelf integrates with [KOReader](koreader.md) for full progress synchronization. You can also connect any OPDS-compatible app via the [OPDS feed](opds.md).
