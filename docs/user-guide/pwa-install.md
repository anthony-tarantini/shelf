# Installing as a PWA

Shelf is a Progressive Web App (PWA) that can be installed on your device for a native app-like experience with offline support.

## Android (Chrome / Chromium)

1. Open your Shelf instance in Chrome.
2. Shelf will display an **install prompt** automatically when available.
3. Alternatively, tap the browser menu (three dots) and select **"Install app"** or **"Add to Home screen"**.
4. Shelf will appear as a standalone app on your home screen.

## iOS (Safari)

iOS Safari does not support automatic install prompts. To install Shelf:

1. Open your Shelf instance in **Safari** (not Chrome or other browsers — only Safari supports PWA installation on iOS).
2. Tap the **Share** button (square with arrow).
3. Scroll down and tap **"Add to Home Screen"**.
4. Confirm the name and tap **"Add"**.

Shelf will appear as a standalone app on your home screen with its own icon.

## Desktop (Chrome / Edge)

1. Open your Shelf instance in Chrome or Edge.
2. Look for the install icon in the address bar, or use the browser menu and select **"Install Shelf"**.
3. Shelf opens in its own window without browser chrome.

## Updates

Shelf updates automatically in the background. When a new version is available, you will see a reload prompt in the app. This is separate from the install flow — you do not need to reinstall to get updates.
